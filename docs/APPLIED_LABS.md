# Applied Labs

Open **Learning Dashboard → Applied Labs**.

The library now contains **36 guided labs** across:

- Power BI, including an optional performance lab
- Excel
- pandas
- analyst communication
- SQL validation and broken-analysis diagnostics
- timed analyst requests
- statistics and experimentation
- funnel, cohort, churn, and variance analysis
- REST API and JSON ingestion
- raw-to-analytics data workflows
- responsible AI-assisted analysis validation

BigQuery remains intentionally excluded.

Each lab includes instructions, starter material, validation criteria,
a dataset folder, a saved submission, roadmap placement, prerequisite
logic, Skills & Concepts evidence, and automatic Demonstrated Evidence.

The new labs are placed across Weeks 4–12 so foundational statistics
precede experiments and regression, business patterns follow core SQL,
API and pipeline work follow pandas and cleaning, responsible-AI work
follows validation and communication, and Power BI performance remains
optional after the core Power BI sequence.
