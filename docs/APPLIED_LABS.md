# Applied Labs

Open **Learning Dashboard → Applied Labs**.

The library contains:

- 6 Power BI labs
- 1 Excel analyst workbook challenge
- 4 pandas bridge labs
- 3 analyst communication labs
- 1 reusable SQL validation lab
- 3 broken-analysis diagnostics
- 3 timed analyst requests

BigQuery exercises are intentionally excluded.

Each lab includes instructions, starter material, validation criteria, a dataset
folder, a saved submission, roadmap placement, status, notes, concept evidence,
and automatic Demonstrated Evidence on completion.

Applied labs are spread across Weeks 3–11 and use lower priority than the
certificate-first core work. Existing compatible Power BI, pandas, and
stakeholder tasks are migrated into the new lab system instead of duplicated.

Saved copies are created under `practice/applied/submissions/`. Power BI, Excel,
and communication labs use Markdown submission records where external artifact
paths can be documented. pandas labs use Python files. SQL-focused labs use SQL
or Markdown depending on the deliverable.
