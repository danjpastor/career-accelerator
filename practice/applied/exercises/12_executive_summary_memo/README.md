# Applied Lab 12: Write an executive summary and stakeholder memo

**Category:** Communication
**Roadmap week:** 5
**Estimated time:** 45 minutes
**Concepts:** problem, finding, impact, recommendation, limitation, audience adaptation

## Objective

Translate analysis into a decision-oriented message for a nontechnical manager.

## Steps

1. Write five sentences: problem, finding, impact, recommendation, limitation.
2. Expand them into a one-page memo.
3. Use one visual only if it improves the decision.
4. Remove unnecessary technical details.
5. End with a requested action.

## Deliverables

- Five-sentence summary.
- One-page memo.
- Note on omitted technical detail.
