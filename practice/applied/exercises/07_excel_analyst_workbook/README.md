# Applied Lab 07: Build an end-to-end Excel analyst workbook

**Category:** Excel
**Roadmap week:** 3
**Estimated time:** 90 minutes
**Concepts:** tables, validation, XLOOKUP, SUMIFS, PivotTables, Power Query, controls

## Objective

Build a controlled workbook that cleans data, calculates reliable metrics, and presents a management summary.

## Steps

1. Load CSV files into named tables.
2. Add data validation and controlled inputs.
3. Use Power Query to clean and combine sources.
4. Use XLOOKUP, SUMIFS, and explicit error handling.
5. Build PivotTables, PivotCharts, slicers, and a one-page summary.
6. Create a controls and assumptions sheet.

## Deliverables

- Finished .xlsx workbook.
- Management-summary screenshot.
- Controls and assumptions sheet.
