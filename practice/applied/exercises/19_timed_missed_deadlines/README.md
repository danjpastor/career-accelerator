# Applied Lab 19: Answer a missed-deadlines request in 30 minutes

**Category:** Timed Analysis
**Roadmap week:** 6
**Estimated time:** 30 minutes
**Concepts:** rapid scoping, SQL analysis, validation, concise recommendation

## Objective

Answer a manager who needs the top causes of missed deadlines within 30 minutes.

## Steps

1. Write one clarifying question.
2. Define missed deadline and grain.
3. Find the top three causes and contribution.
4. Run two checks.
5. Write a three-sentence conclusion and action.

## Deliverables

- Method or SQL.
- Validation results.
- Manager-ready response.
