# Applied Lab 03: Build a star schema and date table

**Category:** Power BI
**Roadmap week:** 8
**Estimated time:** 60 minutes
**Concepts:** fact and dimension tables, relationships, date table, filter direction

## Objective

Model the cleaned data at a clear grain using facts and dimensions.

## Steps

1. Define the grain of shot and time-entry facts.
2. Create project, artist, department, status, and date dimensions.
3. Build one-to-many single-direction relationships.
4. Mark the date table and hide technical fields.

## Deliverables

- Model-view screenshot.
- Relationship inventory.
- Grain statement for each fact.
