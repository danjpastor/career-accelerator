# Retail Operations Performance Dashboard

## Business Problem
A retail operation needs better visibility into sales, labor efficiency, customer satisfaction, returns, promotions, inventory, and profitability.

## Questions
- Which locations and departments perform best?
- How does staffing relate to sales and service?
- When should staffing change?
- Which promotions produce profitable growth?
- What drives returns and poor reviews?
- Where do inventory problems reduce performance?

## Deliverables
- Clean retail dataset
- SQL analysis
- Python EDA
- Power BI dashboards
- Staffing, promotion, and inventory recommendations
