# VFX Production Intelligence Dashboard

## Portfolio Role
Flagship project connecting extensive VFX compositing and supervision experience to formal analytics.

## Business Problem
Production leadership lacks centralized visibility into workload, estimate accuracy, revisions, delivery performance, bottlenecks, and artist utilization.

## Stakeholders
- Head of Production
- VFX Supervisor
- Producers
- Department Leads
- Resource Managers

## Primary KPIs
- On-time delivery rate
- Average turnaround time
- Artist utilization
- Revision rate
- Estimate accuracy
- Hours variance
- Shots completed per week
- Overdue shot count

## Core Questions
1. Which departments miss deadlines most often?
2. Which artists are overloaded?
3. How accurate are initial estimates?
4. Which clients create the most revisions?
5. What percentage of shots are delivered on time?
6. Which projects are highest risk?
7. Where are the main bottlenecks?
8. How does workload change over time?
9. Which departments have the greatest variance?
10. How should resources be reallocated?

## Deliverables
- Synthetic production dataset
- Data dictionary
- SQL schema and analysis
- Python cleaning and EDA notebook
- Power BI executive dashboard
- Power BI workload dashboard
- Executive summary
- GitHub README with screenshots
