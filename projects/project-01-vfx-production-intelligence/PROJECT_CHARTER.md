# Project Charter

## Objective
Build an analytics solution that converts VFX production activity into actionable operational insights.

## Included
- Shot-level production data
- Artist workload
- Estimated and actual hours
- Deadlines and delivery dates
- Revisions
- Project and department performance

## Excluded
- Real confidential studio data
- Individual performance evaluation
- Payroll data

## Success Criteria
- Reproducible data model
- Ten or more business questions answered
- Executive and operational dashboards
- Actionable recommendations
- Professional documentation
