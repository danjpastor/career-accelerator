# Movie Industry Financial Analytics

## Business Problem
A film studio wants to understand which production and release characteristics are associated with stronger financial performance.

## Questions
- Which genres have the strongest ROI?
- How do budgets relate to revenue and profitability?
- Which studios consistently outperform?
- How do ratings, runtime, seasonality, and franchise status relate to returns?
- Where do large budgets show diminishing returns?

## Deliverables
- Combined movie dataset
- SQL financial analysis
- Python exploratory analysis
- Power BI financial dashboard
- Executive recommendations
