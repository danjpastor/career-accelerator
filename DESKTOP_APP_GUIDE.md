# Career Accelerator Desktop App

## Launch

Double-click:

```text
Career Accelerator.bat
```

## Optional Desktop Shortcut

Double-click:

```text
create-desktop-shortcut.vbs
```

A `Career Accelerator` shortcut will be added to the Windows desktop.

## Main Screens

### Dashboard
Shows:
- Current sprint
- Weekly completion
- Study hours
- Google course progress
- SQL problem total
- Active portfolio project
- Next priorities

### Today's Tasks
Use checkboxes to update the active sprint.

### Study Session
Log:
- Date
- Hours
- Google progress
- DataCamp progress
- SQL problems
- Portfolio progress

The app refreshes all progress dashboards automatically.

### Learning
Update:
- Google course
- Google module
- Weekly target hours

### Portfolio
Choose a project and check off project milestones.

### SQL Practice
Create a correctly named `.sql` solution file.

### Weekly Review
Record:
- Biggest win
- Primary blocker
- SQL topic to review
- Next-week adjustment

### Git
- View changed files
- Commit changes
- Push changes

## Recommended Routine

1. Open the app.
2. Review Dashboard and Today's Tasks.
3. Study.
4. Log the study session.
5. Check off completed tasks.
6. Commit changes from the Git tab.
7. Push when ready.

Routine tracking should take less than two minutes.
