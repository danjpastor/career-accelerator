# Demonstrated Evidence

Demonstrated Evidence is a record of proof—not merely a list of topics
studied.

Good evidence names:

1. the skill demonstrated
2. the exact artifact or experience
3. what you personally did
4. how you checked or applied the result

Examples:

- SQL aggregation and HAVING — DuckDB Exercise 02
- Data cleaning and validation — VFX production dataset quality report
- Stakeholder communication — VFX compositing supervision experience
- Power BI modeling — VFX Production Intelligence dashboard

Double-click an evidence item or choose **View Selected** to read its
full description.

Completing a DuckDB exercise through SQL Companion automatically creates
an SQL Practice evidence item.
