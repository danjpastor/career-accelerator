# Job Readiness Layout

Job Readiness uses a fixed page heading and a vertically scrollable
content region.

Protected minimum heights:

- Demonstrated Evidence: 560 px
- Readiness Coach: 300 px
- Highest Leverage: 300 px
- Evidence Coverage: 250 px
- Skills & Concepts: 340 px

Evidence Coverage rows use a fixed 40 px height. The Highest Leverage
action button uses a fixed vertical size policy and a minimum height of
38 px. Readiness Coach provides fallback guidance when no dynamic
recommendation is returned.
